# 🔢🎯 MindNumber

Un juego de adivinanza de números hecho con JavaScript puro — ¡pon a prueba tu intuición y encuentra el número secreto!

> 🌐 [Versión en Inglés](README.md)

## 🌐 Demo en Vivo
👉 [Jugar MindNumber](https://yael123r.github.io/MindNumber/)

## 📸 Vista Previa
![MindNumber Preview](preview.png)

## 🎮 Cómo Jugar
1. Ingresa tu nombre para comenzar
2. Adivina un número entre 1 y 100
3. El juego te dice si el número secreto es mayor o menor
4. ¡Sigue intentando hasta encontrarlo!

## ✨ Características
- 🧠 Generación aleatoria de números
- 📊 Historial de intentos
- 🌙 Modo oscuro / claro
- 👤 Experiencia personalizada con nombre del jugador
- ⌨️ Soporte de teclado (tecla Enter)
